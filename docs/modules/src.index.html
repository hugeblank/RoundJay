Functions

get ()
Gets the entire item index

Returns:

    optional {rj.item.keytable=rj.item.item,...} The list of items internally used by index
getItemFromName (name, dName)
Gets an item from a given Minecraft identifier, and an optional display name.

Parameters:

  • name string The minecraft identifier of the item to get.
  • dName optional string The display name of the item to get.

Returns:

    optional rj.item.item The corresponding item to the given hash, if one exists.
getItemFromHash (hash)
Gets an item that matches the common slot hash of the passed hash string.

Parameters:

  • hash string The slot hash of the item to get.

Returns:

    optional rj.item.item The corresponding item to the given hash, if one exists.

See also:

    src.slot:getHash
findItem (query)
Find an item in the index using fuzzy matching.

Parameters:

  • query string The query to fuzzy match on.

Returns:

    optional rj.item.item The closest item that matched the query, if one exists.
addItem (item)
Add an item to the index.

Parameters:

  • item rj.item.item The item to add.
removeItem (item)
Remove an item from the index. This should generally be used whenever an item has a count of 0, so those items do not display in any listings.

Parameters:

  • item rj.item.item The item to remove.
matchSlotHashes (slots)
From a list of slots, create a hashmap of slots whose keys match.

Parameters:

  • slots {rj.slot.slot,...} An unorganized list of slots.

Returns:

    {string={rj.slot.slot,...},...} A map of slots organized on their hash.

See also:

    getHash
itemsFromHashmap (hashmap)
⚠️ Create items out of a map of slots. This is intended to be used in combination with the above function. Items created by this method directly get added to the index.

Parameters:

  • hashmap {string={rj.slot.slot,...},...} A map of slots organized on their hash.

See also:

    matchSlotHashes
reload ()
⚠️ Reload the item index
completeItemName (input)
Get a list of read-friendly completion strings. Given an input string will return a list of all matches based on both Minecraft Identifier and display name. Case-insensitive.

Parameters:

  • input string An input string, expected to be a partially complete item name.

Returns:

    {string,...} Matches containing the input string, with the part of the string that matched up to input cut off.
