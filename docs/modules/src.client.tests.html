Functions

register (priority, test)
Register a test function that can error correct, or ask for input should something go wrong. Examples can be found in the source.

Parameters:

  • priority number The priority of this test, lower number = higher priority. Must be greater than 1.
  • test function The test function to run. Must return true/false for pass/fail.
runTests ()
Run all registered tests. This function is meant to be used by clients and scripts before user input is prompted, or script is ran.
