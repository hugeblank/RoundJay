new (inventory, nSlot, details)
Creates a slot object.

Parameters:

  • inventory string The inventory this slot resides in.
  • nSlot int The slot number of the inventory this slot represents.
  • details table The basic item details of this slot.

Returns:

    Slot A slot object representing the parameters given.

Class slot

Class whose objects represents a single item stack in a slot in an inventory.
slot:getHash ()
Get the unique hash of the item contained in a slot. This is not the same as the nbt hash, but rather a combination of the item identifier and nbt hash, if it exists.

slot:take (to, amount, tSlot)
⚠️ Take items from this slot.

Parameters:

  • to string The inventory the items should be put into.
  • amount int The amount of items to be taken from the slot.
  • tSlot int The optional slot id of the `to` inventory the items should be put into.

Returns:

    int The number of items taken from the slot.
slot:put (from, fSlot)
⚠️ Put items into this slot.

Parameters:

  • from string The inventory from which the items should be brought in from.
  • fSlot int The slot id of the `from` inventory in which the targeted items exist.

Returns:

    int the number of items put into the slot.
