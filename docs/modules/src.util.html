Functions

getNames ()
⚠️ Get a list of peripheral names on the pool network that can act as storage for items. Does not contain interface chest, blacklisted inventories, as well as any non-chest/non-shulker box inventories.

Returns:

    {string,...} Valid inventory peripheral names.
getRandomEmptySlots (amount)
⚠️ Get a list of random empty slots.

Parameters:

  • amount int The amount of slots to return.

Returns:

    {emptySlot,...}
getRandomEmptySlot ()
⚠️ Get a singular random empty slot.

Returns:

  1. string The inventory to which this slot belongs.
  2. int The index of this slot within the inventory.
getFreeSpace ()
⚠️ Get a total of the remaining free slots in the storage pool, and the total slots in the system.

Returns:

  1. int empty The total number of free slots remaining in the system.
  2. int slots The total number of all slots in the system.
transferAndSlot (from, fslot, details, to, tslot)
⚠️ Transfers items to a slot, then allocates a new Slot object for them. Parameters are identical to item objects store method.

Parameters:

  • from string The inventory from which to pull items from.
  • fslot string The slot in `from`.
  • details rj.slot.basicDetails The basic information about the slot being pulled from.
  • to optional string The inventory to put the items into.
  • tslot optional int The slot in `to`.

Returns:

    rj.slot.slot The slot that the items were stored into.

See also:

    src.item.store
closestMatch (query, data)
Get the value of the closest match of a query against keys in a table.

Parameters:

  • query string A query string to compare to the keys of the data table.
  • data {string=any} A table containing string keys.

Returns:

    any The value of the string that was found to be the closest match in the data table.
parseAmount (item, amount)
Parses an amount string to determine the intended amount of items from a given item to handle. If the string "stack" is passed into amount, then the stack size of the item will be used for amount. If the string "all" is passed into amount, then the total amount of items will be used for amount.

Parameters:

  • item rj.item.item The item to parse amount on, if necessary.
  • amount string The numerical amount of items, "stack", or "all".

Returns:

    int The amount of items to be used.

Tables

emptySlot
A table representing an empty slot in the storage pool. This is the precursor to what will eventually become a proper slot. We use this as an intermediary index to push the items, then we turn it into a slot object.

Fields:

  • chest string The inventory to which this slot belongs.
  • slot int The index of this slot within the inventory.
