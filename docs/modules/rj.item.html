take (location, count, slot)
⚠️ Take items from the pool of slots that this item represents

Parameters:

  • location string The destination inventory where the items should go
  • count int The amount of items to take. If greater than the total amount of items, will pull from all slots.
  • slot int The slot of the destination inventory where items should go.

Returns:

    int The amount of items taken.
store (from, fslot, fdetails, to, tslot)
⚠️ Store all items from a slot into an inventory/slot pair If repeatedly calling, consider allocating `to` and `tslot` using `util.getRandomEmptySlots`.

Parameters:

  • from string The inventory from which to pull items from.
  • fslot string The slot in `from`.
  • fdetails rj.slot.basicDetails The basic information about the slot being pulled from.
  • to optional string The inventory to put the items into.
  • tslot optional int The slot in `to`.
