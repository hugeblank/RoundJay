rj.slot:Slot (inventory, slot, details)
Creates a slot object. This function is what gets returned when requiring `rj.slot`.

Parameters:

  • inventory string The inventory this slot resides in.
  • slot int The slot number of the inventory this slot represents.
  • details table The basic item details of this slot.
rj.slot:getHash ()
Get the unique hash of the item contained in a slot. Note: this is not the same as the nbt hash, but rather a combination of the item identifier and nbt hash, if it exists.
